how to create a presentation in shadowpoint

- created by shadowdev -

if you want to find something specific use control + f

cof stands for config btw


how to create a presentation directory

create a folder named after the presentation


how to create a slide

to create a slide you will have to make a folder named the slide number

for example the first slide is gonna be named 1 and the second one is gonna be 2


how to change the title and the background and etc

create a file named config.cof in your slide folder

to change a background add a picture to the folder for example bg.png

and write to the second line \bg.png


if you want to change the title of the presentation window

then youl have to open the config.cof and write the window title that you want to use on the first line


if you also want to maybe disable the form icon

then just go to the third line of config.cof and write true there to disable it



thats all for config.cof


how to add text and pictures to slides


first youre gonna need to create a file named slide.cof

and on the first line youre gonna need to put there how many objects

(an object is a picture or a text) are gonna be on the slide

and then for the lines after it youl have to reference the objects



to reference an object youre gonna have to just write the cof file name like text1.cof to the slide.cof
